# MVP Scope — V1

## In Scope
- [ ] 3 service categories: Plombier, Électricien, Peintre
- [ ] Tunis governorate only
- [ ] Artisan browse + filter
- [ ] Artisan profile view
- [ ] 2-step booking request flow
- [ ] Artisan availability toggle
- [ ] Artisan request accept/decline
- [ ] WhatsApp contact bridge
- [ ] Post-job rating (1–5 stars)
- [ ] Cash payment (no gateway)

## Out of Scope (V2+)
- Payment gateway (D17, Flouci)
- In-app messaging
- Real-time location/GPS
- Push notifications (FCM)
- Admin panel
- Onboarding / auth flows
- Backend / database
- More service categories
- Sfax, Sousse expansion

## Success Metrics (Month 3)
- 60+ verified artisans onboarded
- 50+ completed bookings
- <30 min average artisan response time
- 4.0+ average artisan rating
